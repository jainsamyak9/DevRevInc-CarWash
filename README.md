 
# WEB Application Car Wash 

This is on demand car wash service
with the following tools.

- Eclipse
- MySQL Workbench

### MySQL Tables
- User Cred
- Admin Cred
- Place Details
- Request Booking
- Available Slots

### Some Details
- Admin Controls
 - Create Time Slots
 - Accept/ Reject Booking
 - View Bookings
 - login
 - logout
- User Controls
 - Login
 - SignUp
 - Search based on place
 - Booking Availabilty
 - Get Status
 - Logout
